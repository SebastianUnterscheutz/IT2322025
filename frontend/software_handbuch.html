Handbuch zur Mitfahrgelegenheiten-Web-Applikation

Fahrgemeinschaften für den Weg zum BSZ Vogtland in Rodewisch

Einleitung

Dieses Handbuch beschreibt die Nutzung und Funktionalitäten der Fahrgemeinschafts-Web-Applikation, die es Benutzern ermöglicht, Mitfahrgelegenheiten anzubieten oder zu suchen, speziell für den Weg zum BSZ Vogtland in Rodewisch.

Die Software bietet eine benutzerfreundliche Oberfläche zur Eingabe und Suche von Fahrgemeinschaften sowie zur Anzeige von Mitfahrgelegenheiten auf einer interaktiven Karte. Sie ermöglicht den Benutzern, sowohl als Mitfahrer als auch als Fahrer mitzufahren und eine geeignete Fahrgelegenheit zu finden.

Systemanforderungen

Bevor Sie die Anwendung nutzen können, stellen Sie bitte sicher, dass folgende Voraussetzungen erfüllt sind:

  • Internetbrowser: Chrome, Firefox, Safari
  • Keine speziellen Installationen erforderlich

Funktionalitäten

Mitfahrgelegenheit anbieten

Beschreibung: Diese Funktion ermöglicht es den Nutzern, eine Mitfahrgelegenheit anzubieten.

Formularfelder:

  • PLZ: (Pflichtfeld) Geben Sie Ihre Postleitzahl ein.
  • Ort: (Pflichtfeld) Geben Sie den Ort ein, von dem Sie abfahren.
  • Straße, Hausnummer: (Optional) Geben Sie die Straße und Hausnummer ein.
  • Name: (Pflichtfeld) Geben Sie Ihren Nachnamen ein.
  • Vorname: (Optional) Geben Sie Ihren Vornamen ein.
  • E-Mail: (Pflichtfeld) Geben Sie Ihre E-Mail-Adresse ein.
  • Klasse: (Optional) Geben Sie Ihre Klasse ein (z. B. IT23x).
  • Handynummer: (Optional) Geben Sie Ihre Handynummer ein.
  • Gültigkeitsdatum von: (Optional, Standard: ab sofort) Das Startdatum der Mitfahrgelegenheit.
  • Gültigkeitsdatum bis: (Optional, Standard: 6 Monate) Das Enddatum der Mitfahrgelegenheit.
  • Weitere Informationen: (Optional) Geben Sie Zusatzinformationen wie Kostenbeteiligung ein.

Funktionsweise: Wenn das Formular erfolgreich ausgefüllt ist, wird eine Mitfahrgelegenheit erstellt und in der Datenbank gespeichert. Ein Zufallscode wird generiert, um das Angebot später zu bearbeiten oder zu löschen. Der Benutzer erhält eine Bestätigung per E-Mail.

Mitfahrgelegenheit suchen

Beschreibung: Diese Funktion erlaubt es den Nutzern, nach Mitfahrgelegenheiten zu suchen.

Eingabefelder:

  • PLZ: (Pflichtfeld) Geben Sie Ihre Postleitzahl ein.
  • Ort: (Pflichtfeld) Geben Sie den Ort ein, in dem Sie eine Mitfahrgelegenheit suchen.

Kartenanzeige: Nach der Eingabe der Postleitzahl und des Ortes werden alle Mitfahrgelegenheiten auf einer Karte angezeigt. Durch Klicken auf einen Pin auf der Karte erhalten Sie weitere Informationen zu der Mitfahrgelegenheit, wie Name des Anbieters, Startzeit und -ort sowie eine Kontaktmöglichkeit.

Erweiterte Funktionen

  • Umkreissuche: Suche nach Mitfahrgelegenheiten im Umkreis von z.B. 5 km Luftlinie.
  • Liste der Mitfahrgelegenheiten: Darstellung der Mitfahrgelegenheiten als Liste mit den wichtigsten Informationen.
  • Schutz gegen Fake-Angebote: Die Applikation ist gegen massenhafte Fake-Angebote geschützt. Falsche oder unvollständige Angaben werden abgefangen und können vom Support bearbeitet werden.

Technische Details

Datenbankstruktur

  • Mitfahrgelegenheiten: Tabelle mit den Feldern (PLZ, Ort, Straße, Hausnummer, Name, E-Mail, etc.).
  • Benutzerdaten: Speicherung von Informationen für das Bearbeiten der Angebote.
  • Umkreissuche: Implementierung einer geografischen Suche für die Mitfahrgelegenheiten.

Technologie-Stack

  • Frontend: HTML, CSS, JavaScript, OpenStreetMap API für die Kartenanzeige.
  • Backend: PHP mit MySQL-Datenbank für die Speicherung der Mitfahrgelegenheiten.
  • Benutzerdaten: Sicherheit durch Verschlüsselung von persönlichen Daten.

Fehlerbehandlung und Eingabekontrollen

  • Pflichtfelder: Alle Pflichtfelder müssen vor dem Absenden ausgefüllt werden.
  • Datenvalidierung: Eine E-Mail-Adresse muss im richtigen Format eingegeben werden. Alle Zahlen müssen numerisch sein, und die Postleitzahl muss eine gültige Formatierung haben.
  • Fehlermeldungen: Bei falschen Eingaben zeigt das System hilfreiche Fehlermeldungen, um den Benutzer bei der Korrektur zu unterstützen.

Nutzerhilfe und Support

Für technische Probleme oder Anfragen zum Anlegen und Bearbeiten von Mitfahrgelegenheiten steht unser Support-Team zur Verfügung. Sie können uns per E-Mail oder über das Kontaktformular erreichen.

Design und User Experience

Die Anwendung ist so gestaltet, dass sie eine einfache und benutzerfreundliche Navigation ermöglicht. Die interaktive Karte bietet eine klare Darstellung der verfügbaren Mitfahrgelegenheiten. Durch klare Felder und erklärende Texte wird der Nutzer durch den Prozess geführt, ohne dass er auf zusätzliche Hilfe angewiesen ist.

Sicherheit und Datenschutz

  • Datenverschlüsselung: Alle persönlichen Daten werden sicher verschlüsselt gespeichert.
  • Zugangskontrolle: Jeder Benutzer kann nur seine eigenen Mitfahrgelegenheiten bearbeiten oder löschen.

Anwendungsfalldiagramm und Aktivitätsdiagramm

Das Anwendungsfalldiagramm und das Aktivitätsdiagramm visualisieren den Ablauf und die Interaktionen zwischen den Nutzern und der Applikation. Diese Diagramme sollten in einem UML-Dokument erstellt werden und beinhalten:

  • Anwendungsfalldiagramm: Zeigt die wichtigsten Anwendungsfälle (Mitfahrgelegenheit anbieten, Mitfahrgelegenheit suchen, Benutzerdaten verwalten).
  • Aktivitätsdiagramm: Zeigt die Ablauflogik für das Anbieten und Suchen von Mitfahrgelegenheiten.
